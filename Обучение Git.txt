1 Создать аккаунт на GitHub
2 Создать репозиторий на GitHub
3 Скачать SourceTree
4 В SourceTree жмем на кнопку Refresh OAuthToken, и оно автоматически засинхронизирует SourceTree с акаунтом GitHub
5 Если НЕ свяжет то наверное нужно сделать Personal access token (Но поидее не нужно.)
6 В SourceTree жмем Remote, там находим репозитории нашего GitHub-a, клонируем
7 В SourceTree жмем кнопку terminal, оно откроет консоль git-bash (Не открыло git-bash, то нужно его скачать) https://gitforwindows.org/
8 Нужно проинициализировать GitFlow. В терминале пишем команду (git flow init). Оно попросит выбрать главную ветку - пишем main, остальные ветки называть по умолчанию (кнопка Enter)
9 Теперь нам доступен функционал Git Flow.
10 И теперь можно спокойно использовать git в связке с SourceTree.
